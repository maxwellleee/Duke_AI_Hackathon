# Duke AI Hackathon - Gesturify
Gesturify is a web app that helps people learn sign language in real time using a machine learning model that reads their camera!
You can:
Practice ASL letters and words with modules
Test your knowledge in time based word challenges
Compete with other users and try to beat top scores
